# Complete Data Analysis Project

## Project Overview

This project analyzes an e-commerce sales dataset using Python, Pandas, and Matplotlib. The program loads the dataset, performs data analysis, and creates visualizations to understand sales performance.

## Features

- Load CSV data
- Explore dataset
- Check missing values
- Calculate sales metrics
- Find best-selling product
- Generate bar chart
- Generate pie chart

## Requirements

- Python 3.x
- Pandas
- Matplotlib

## Run

pip install -r requirements.txt

python main.py
