Complete Data Analysis Project Report

Project Overview

The objective of this project is to analyze an e-commerce sales dataset using Python. The project includes loading data, exploring the dataset, performing calculations, creating visualizations, and presenting useful business insights.

What I Learned

This project helped me understand the complete data analysis workflow. I learned how to load datasets, clean and analyze data using Pandas, and create charts using Matplotlib. I also learned how to present the results in a clear and organized format.

Setup Instructions

1. Install Python.
2. Install required libraries:

pip install -r requirements.txt

3. Keep the following folder structure:

data/
visualizations/
report/

4. Run:

python main.py

Code Structure

main.py
Contains the complete analysis and visualization code.

data/
Contains the sales dataset.

visualizations/
Contains the generated charts.

report/
Contains the project documentation.

Technical Details

The project uses:

- Pandas for data analysis
- Matplotlib for visualization

The following tasks were completed:

- Loaded dataset
- Checked missing values
- Calculated total revenue
- Calculated average sales
- Found highest sale
- Identified best-selling product
- Created bar chart
- Created pie chart
